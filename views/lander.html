This application uses machine learning object identificaiton as "authenticaiton". This example was built with ml5's ObjectDetector() along with CocoSsd to identify objects defined in the COCO dataset, as well as the locations of those objects in the frame. Once the user displays his/herself (person) and the speicifed object, they are routed to the homepage of the website. If you are on a mobile device this feature is currently unsupported, please press "Skip :(" To allow access to your camera for detection, please use HTTPS (https://kerrickc.herokuapp.com).
